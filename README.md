<h1>Emirs Miniprosjekt :sunglasses:

<h2>Hvordan vise siden

<h3>For å kjøre siden trenger man å bygge klienten, deretter kjøre serveren.

<h3> 1. Bygge klienten
```sh
cd Klient
npm install
npm run build
cd ..
```

<h3> 2. Kjøre serveren
```sh
cd Server
npm install
npm start
```

<h3> Nå er siden tilgjengelig på localhost:8080
