<h1>Emirs Miniprosjekt :sunglasses:

<h2>Hvordan vise siden

<h3>For å kjøre siden trenger man å bygge klienten, deretter kjøre serveren.


1. Bygge klienten:

```sh
cd Klient
npm install
npm run build
cd ..
```

2. Kjøre serveren:

```sh
cd Server
npm install
npm start
```

<h3> Nå er siden tilgjengelig på 
http://localhost:8080/

<h2>Hvordan teste siden


1. Teste klienten:

```sh
cd Klient
npm install
npm test
npm run flow
cd ..
```

2. Teste serveren:

```sh
cd Server
npm install
npm test
npm run flow
```
